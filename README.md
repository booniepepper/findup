findup 1.0

USAGE:
    findup FILE

FLAGS:
    -h, --help    Prints help information
    -V, --version Prints version information

Finds a directory containing FILE. Tested by filename with exact string equality. Starts searching at the current working directory and recurses "up" through parent directories.

The first directory containing FILE will be printed. If no directory contains FILE, nothing is printed and the program exits with an exit code of 1.‚èé
